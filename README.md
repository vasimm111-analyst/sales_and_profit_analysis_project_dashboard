# sales_and_profit_analysis_project_dashboard

## Project Overview
This project is an interactive Sales Dashboard created in Microsoft Excel to analyze business sales performance.

The dashboard provides insights into:
- Sales trends
- Profit analysis
- Customer performance
- State-wise sales
- Category-wise sales

---

## Features
- Interactive Year and Category Filters
- Monthly Sales Analysis
- Profit Trend Visualization
- Top 5 Customers Analysis
- State-wise Sales Map
- Customer Count Tracking
- Category-wise Sales Comparison

---

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Map Charts
- Data Cleaning

---

## Dashboard Insights
- Chairs generated the highest sales.
- California showed the highest sales contribution.
- Sales increased significantly in the last months.
- Top customers contributed a major portion of revenue.

---

## Business Benefits
- Helps identify high-performing products
- Tracks sales growth over time
- Supports better business decisions
- Improves customer analysis
